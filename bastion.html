About the Game

Bastion is a roguelite tower defense game built from the ground up in Godot 4. Players strategically place defensive units across procedurally generated levels, managing limited resources against escalating waves of enemies.

Each run is different — level layouts shift, enemy compositions vary, and the units available to you change. The goal is a game that rewards learning and adaptation over memorization.

Development

This project serves as both a creative outlet and a serious technical exercise — exploring Godot 4's capabilities, procedural generation, 3D asset pipelines via Meshy AI, and game design fundamentals all at once.
